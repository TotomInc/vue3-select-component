## Prerequisites

[Vue.js](https://vuejs.org) **3.5+** is required.

## Install the package

::code-group
```bash [pnpm]
pnpm add vue3-select-component
```

```bash [npm]
npm install vue3-select-component
```

```bash [yarn]
yarn add vue3-select-component
```

```bash [bun]
bun add vue3-select-component
```
::

## Import paths

v1 exposes two entry points:

| Import | Use case |
|--------|----------|
| `vue3-select-component` | Assembled `Select` component (default export) |
| `vue3-select-component/primitives` | Low-level primitive components |
| `vue3-select-component/styles` | Default assembled Select styles (`--vs-*` variables) |
| `vue3-select-component/styles.css` | Same stylesheet (explicit `.css` subpath) |

```ts
// Batteries-included
import Select from "vue3-select-component"
import "vue3-select-component/styles"

// Headless composition
import {
  SelectRoot,
  SelectTrigger,
  SelectValue,
  SelectPopover,
  SelectListbox,
  SelectOption,
} from "vue3-select-component/primitives"
```

## Styling

Import default styles for the assembled `Select`, then customize via `--vs-*` CSS variables (see [Styling](/guide/styling)).

Primitives are headless. Use the default stylesheet or write your own CSS targeting `data-*` hooks.
