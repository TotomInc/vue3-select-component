The assembled `Select` wires up all primitives with sensible defaults. It is the fastest way to get started.

## Use single select

Try the assembled `Select` in single-select mode:

::client-only
:usage-single-select-demo

#fallback
:select-demo-fallback
::

Import the component and bind `v-model` to your selected value:

```vue [App.vue]
<script setup lang="ts">
import { ref } from "vue"
import Select from "vue3-select-component"
import "vue3-select-component/styles"

const selected = ref<string | null>(null)
</script>

<template>
  <Select
    v-model="selected"
    :options="[
      { label: 'Option #1', value: 'option_1' },
      { label: 'Option #2', value: 'option_2' },
      { label: 'Option #3', value: 'option_3' },
    ]"
    placeholder="Select an option"
  />
</template>
```

Your IDE provides autocompletion and type checking automatically.

## Use multi-select

Try multi-select with the `multiple` prop:

::client-only
:usage-multi-select-demo

#fallback
:select-demo-fallback
::

Bind `v-model` to an array:

```vue [App.vue]
<script setup lang="ts">
import { ref } from "vue"
import Select from "vue3-select-component"
import "vue3-select-component/styles"

const selected = ref<string[]>([])
</script>

<template>
  <Select
    v-model="selected"
    multiple
    :options="[
      { label: 'Option #1', value: 'option_1' },
      { label: 'Option #2', value: 'option_2' },
      { label: 'Option #3', value: 'option_3' },
    ]"
  />
</template>
```

::note
Upgrading from v0? Rename boolean props (`isMulti` → `multiple`, `isSearchable` → `searchable`, etc.). See the [migration guide](/guide/migration).
::

## Disable search or add clear

Search is enabled by default on the assembled `Select`. Disable it with `:searchable="false"` when you want a button-only select. Add `clearable` to show a clear button when a value is selected.

::client-only
:usage-clearable-demo

#fallback
:select-demo-fallback
::

```vue [App.vue]
<script setup lang="ts">
import { ref } from "vue"
import Select from "vue3-select-component"
import "vue3-select-component/styles"

const selected = ref<string | null>(null)
</script>

<template>
  <Select
    v-model="selected"
    :options="options"
    clearable
    placeholder="Search and select"
  />
</template>
```

## Bind data with v-model

The selected value stays reactive via Vue's `v-model`.

| Mode | `v-model` type | Default |
|------|----------------|---------|
| Single | `string \| number \| null` | `null` |
| Multi | `(string \| number)[]` | `[]` |

[Learn more about `v-model`](https://vuejs.org/guide/components/v-model.html).
