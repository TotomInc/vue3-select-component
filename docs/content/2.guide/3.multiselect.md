Multi-select lets users pick more than one option. Enable it with the `multiple` prop on assembled `Select` or `SelectRoot`.

## Enable multi-select

```vue [App.vue]
<script setup lang="ts">
import { ref } from "vue"
import Select from "vue3-select-component"
import "vue3-select-component/styles"

const selected = ref<string[]>([])
</script>

<template>
  <Select
    v-model="selected"
    multiple
    :options="[
      { label: 'JavaScript', value: 'js' },
      { label: 'TypeScript', value: 'ts' },
      { label: 'Vue', value: 'vue' },
    ]"
    placeholder="Select languages"
  />
</template>
```

## Understand model behavior

| Aspect | Behavior |
|--------|----------|
| `v-model` type | `(string \| number)[]` |
| Default value | `[]` |
| Selecting | Adds value to the array. Menu stays open. |
| Deselecting | Click a selected option to toggle it off |
| Clear button | Resets to `[]`, emits `optionDeselected(null)` |

## Selected option checkmarks

In multi-select mode, selected options in the dropdown show a checkmark on the right. The assembled `Select` and primitive `SelectOption` render this by default. Override it with the `checkmark` slot on `SelectOption`.

```vue [CustomSelect.vue]
<script setup lang="ts">
import { ref } from "vue"
import {
  SelectInput,
  SelectListbox,
  SelectNoOptions,
  SelectOption,
  SelectPopover,
  SelectRoot,
  SelectTrailingIcon,
  SelectTrigger,
  SelectValue,
} from "vue3-select-component/primitives"
import "vue3-select-component/styles"

const selected = ref<string[]>([])

const options = [
  { label: "JavaScript", value: "js" },
  { label: "TypeScript", value: "ts" },
  { label: "Vue", value: "vue" },
]
</script>

<template>
  <div data-assembled-select>
    <SelectRoot v-model="selected" multiple searchable>
      <SelectTrigger>
        <SelectValue placeholder="Pick languages" />
        <SelectInput />
        <SelectTrailingIcon />
      </SelectTrigger>

      <SelectPopover>
        <SelectListbox>
          <SelectNoOptions />
          <SelectOption
            v-for="opt in options"
            :key="opt.value"
            :value="opt.value"
            :label="opt.label"
          >
            <template #checkmark>
              <UIcon name="i-lucide-check" />
            </template>
          </SelectOption>
        </SelectListbox>
      </SelectPopover>
    </SelectRoot>
  </div>
</template>
```

Style the layout with `data-select-option-label` and `data-select-option-checkmark`.

## Hide selected options

By default, selected options stay visible in the dropdown so users can deselect them. Set `hide-selected` on assembled `Select` to remove them from the list:

```vue [App.vue]
<script setup lang="ts">
import { ref } from "vue"
import Select from "vue3-select-component"
import "vue3-select-component/styles"

const selected = ref<string[]>(["js"])

const options = [
  { label: "JavaScript", value: "js" },
  { label: "TypeScript", value: "ts" },
  { label: "Vue", value: "vue" },
]
</script>

<template>
  <Select
    v-model="selected"
    multiple
    hide-selected
    :options="options"
    placeholder="Select languages"
  />
</template>
```

On primitives, pass `hide-selected` to `SelectRoot`:


## Customize tag remove icon

In multi-select mode, each tag shows a remove button. Override the icon on the assembled `Select` with the `tag-remove` slot:

```vue [App.vue]
<script setup lang="ts">
import { ref } from "vue"
import Select from "vue3-select-component"
import "vue3-select-component/styles"

const selected = ref<string[]>(["js"])

const options = [
  { label: "JavaScript", value: "js" },
  { label: "TypeScript", value: "ts" },
  { label: "Vue", value: "vue" },
]
</script>

<template>
  <Select v-model="selected" multiple :options="options">
    <template #tag-remove>
      <UIcon name="i-lucide-x" />
    </template>
  </Select>
</template>
```

The slot receives `{ label, value }` for each tag. On primitives, use the `remove` slot on `SelectTag` or forward it from `SelectValue` with `tag-remove`.

## Display selected values

By default, `SelectValue` renders a `SelectTag` for each selection. Customize tags with primitive slots:

```vue [CustomSelect.vue]
<script setup lang="ts">
import { ref } from "vue"
import {
  SelectInput,
  SelectListbox,
  SelectNoOptions,
  SelectOption,
  SelectPopover,
  SelectRoot,
  SelectTag,
  SelectTrailingIcon,
  SelectTrigger,
  SelectValue,
} from "vue3-select-component/primitives"
import "vue3-select-component/styles"

const model = ref<string[]>([])

const options = [
  { label: "JavaScript", value: "js" },
  { label: "TypeScript", value: "ts" },
  { label: "Vue", value: "vue" },
]
</script>

<template>
  <div data-assembled-select>
    <SelectRoot v-model="model" multiple :options="options" searchable>
      <SelectTrigger>
        <SelectValue placeholder="Pick languages">
          <template #default="{ selectedOptions }">
            <SelectTag
              v-for="opt in selectedOptions"
              :key="String(opt.value)"
              :value="opt.value"
              :label="opt.label"
            >
              <template #default="{ label }">
                <strong>{{ label }}</strong>
              </template>
            </SelectTag>
          </template>
        </SelectValue>
        <SelectInput />
        <SelectTrailingIcon />
      </SelectTrigger>

      <SelectPopover>
        <SelectListbox>
          <SelectNoOptions />
          <SelectOption
            v-for="option in options"
            :key="option.value"
            :value="option.value"
            :label="option.label"
          />
        </SelectListbox>
      </SelectPopover>
    </SelectRoot>
  </div>
</template>
```

## Remove values with keyboard

When the menu is open, searchable, and the search field is empty, **Backspace** removes the last selected tag. This updates `v-model` without emitting `optionDeselected`.

## Combine with search and clear

```vue [App.vue]
<script setup lang="ts">
import { ref } from "vue"
import Select from "vue3-select-component"
import "vue3-select-component/styles"

const selected = ref<string[]>([])

const options = [
  { label: "JavaScript", value: "js" },
  { label: "TypeScript", value: "ts" },
  { label: "Vue", value: "vue" },
]
</script>

<template>
  <Select
    v-model="selected"
    multiple
    clearable
    :options="options"
    placeholder="Search and select languages"
  />
</template>
```

- **Search** filters visible options without affecting selected values
- **Clear** removes all selections at once
