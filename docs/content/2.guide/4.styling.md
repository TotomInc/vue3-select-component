v1 primitives ship **unstyled**. Style them with `data-*` attribute selectors.

::warning
v0 shipped bundled CSS via `import "vue3-select-component/styles"` with `--vs-*` variables. v1 removes bundled styles in favor of headless primitives. See the [migration guide](/guide/migration) when upgrading.
::

## Target data attributes

Every primitive exposes stable `data-*` hooks:

| Attribute | Element | State |
|-----------|---------|-------|
| `data-select-root` | Root wrapper | |
| `data-assembled-select` | Assembled root | |
| `data-select-trigger` | Trigger button | |
| `data-select-value` | Value display | |
| `data-select-input` | Search input | |
| `data-select-popover` | Dropdown panel | `aria-hidden` |
| `data-select-listbox` | Listbox container | |
| `data-select-option` | Option row | `data-value`, `data-active` |
| `data-select-tag` | Multi-value tag | |
| `data-select-tag-remove` | Tag remove button | |
| `data-select-clear` | Clear button | |
| `data-select-indicator` | Chevron / loader | `data-open`, `data-loading` |
| `data-select-no-options` | Empty state | |

## Style the trigger

```css
[data-select-trigger] {
  display: flex;
  align-items: center;
  gap: 0.5rem;
  width: 100%;
  min-height: 2.375rem;
  padding: 0.25rem 0.5rem;
  border: 1px solid #e4e4e7;
  border-radius: 0.375rem;
  background: #fff;
  cursor: pointer;
}

[data-select-trigger]:disabled {
  opacity: 0.5;
  cursor: not-allowed;
}
```

## Style options

Use `data-active` for keyboard-highlighted options and `aria-selected` for selected state:

```css
[data-select-option] {
  padding: 0.5rem 0.75rem;
  cursor: pointer;
}

[data-select-option][data-active="true"] {
  background: #dbeafe;
}

[data-select-option][aria-selected="true"] {
  background: #93c5fd;
}

[data-select-option][aria-disabled="true"] {
  opacity: 0.5;
  cursor: not-allowed;
}
```

## Style the popover

```css
[data-select-popover] {
  margin-top: 0.5rem;
  border: 1px solid #e4e4e7;
  border-radius: 0.375rem;
  background: #fff;
  box-shadow: 0 1px 2px rgb(0 0 0 / 0.05);
  z-index: 50;
}
```

## Style multi-value tags

```css
[data-select-tag] {
  display: inline-flex;
  align-items: center;
  gap: 0.25rem;
  padding: 0.125rem 0.5rem;
  border-radius: 0.25rem;
  background: #f4f4f5;
  font-size: 0.75rem;
}

[data-select-tag-remove] {
  border: none;
  background: transparent;
  cursor: pointer;
  line-height: 1;
}
```

## Style the indicator

```css
[data-select-indicator][data-open="true"] {
  transform: rotate(180deg);
}

[data-select-indicator][data-loading="true"]::after {
  content: "";
  display: inline-block;
  width: 1rem;
  height: 1rem;
  border: 2px solid currentColor;
  border-top-color: transparent;
  border-radius: 50%;
  animation: spin 0.6s linear infinite;
}
```

## Scope styles to a wrapper

Wrap your select in a container class to scope styles:

```vue
<template>
  <div class="my-select">
    <Select v-model="value" :options="options" />
  </div>
</template>

<style scoped>
.my-select [data-select-trigger] {
  border-radius: 12px;
}
</style>
```

For primitives, scope via a parent class or use `:deep()` as needed.
