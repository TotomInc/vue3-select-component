Vue 3 Select Component follows the [WAI-ARIA Combobox Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/combobox/) for assistive technologies.

## Built-in features

- **ARIA attributes** for semantic markup (`combobox`, `listbox`, `option`)
- **Keyboard navigation** for full keyboard-only operation
- **Focus management** with roving `aria-activedescendant` on the trigger
- **Screen reader support** with descriptive labels and state attributes

## ARIA roles and attributes

### Trigger

The trigger is a `<button>` with combobox semantics when searchable:

| Attribute | Purpose |
|-----------|---------|
| `aria-expanded` | Menu open (`true`) or closed (`false`) |
| `aria-haspopup="listbox"` | Signals a listbox popup |
| `aria-controls` | References the listbox element ID |
| `role="combobox"` | Applied when `searchable` is enabled |
| `aria-activedescendant` | Points to the keyboard-focused option |

### Listbox

```html
<div role="listbox" aria-multiselectable="true|false" tabindex="-1">
```

`aria-multiselectable` is `true` in multi-select mode.

### Options

Each option renders as:

```html
<div role="option" aria-selected="true|false" aria-disabled="true|false">
```

Disabled options cannot be selected or keyboard-focused.

### Interactive controls

| Element | Accessible name |
|---------|-----------------|
| Clear button | `aria-label="Clear selection"` |
| Tag remove button | `aria-label="Remove"` |
| Indicator | `aria-hidden="true"` (decorative) |
| Empty state | `role="status"` |

## Keyboard interactions

| Key | Menu closed | Menu open |
|-----|-------------|-----------|
| `ArrowDown` / `ArrowUp` | Open menu | Move active option |
| `Space` (no search text) | Open menu | Select active option |
| `Enter` | | Select active option |
| `Escape` | | Close menu |
| `Tab` | | Close menu |
| `Backspace` (multi, empty search) | | Remove last selected tag |

Single-select closes the menu after selection. Multi-select keeps it open so users can pick multiple values.

## Dismiss behavior

- Click outside the select closes the menu
- Tab away closes an open menu

## Label your select

Associate the select with a visible label using standard HTML:

```vue
<template>
  <label id="country-label" for="country-select">Country</label>
  <Select
    id="country-select"
    v-model="country"
    :options="countries"
    aria-labelledby="country-label"
  />
</template>
```

For primitives, apply `aria-labelledby` on `SelectTrigger` via a wrapper or by extending the trigger.

## Test with assistive technology

Verify these scenarios during development:

1. Open and close the menu with keyboard only
2. Navigate options with arrow keys and select with Enter
3. Confirm screen reader announces expanded/collapsed state
4. Verify disabled options are skipped
5. In multi-select, confirm tags are announced and removable
