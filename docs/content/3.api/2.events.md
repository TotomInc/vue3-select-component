## Assembled Select events

Use kebab-case in templates (`@option-selected`) or camelCase in script (`@optionSelected`).

### optionSelected

Emitted when an option is selected, in the same tick as the `v-model` update.

| | |
|---|---|
| **Payload** | `GenericOption`, the **source** option object |

```vue
<Select
  v-model="selected"
  :options="options"
  @option-selected="(option) => console.log(option.label, option.value)"
/>
```

::tip
For tracking the current selection, prefer a `computed` derived from `v-model` over listening to this event:

```ts
const selectedOption = computed(() =>
  options.find((o) => o.value === selected.value),
)
```
::

### optionDeselected

Emitted when an option is deselected.

| | |
|---|---|
| **Payload** | `GenericOption \| null` |

`null` is emitted when clearing all values in multi-select mode.

```vue
<Select
  v-model="selected"
  :options="options"
  @option-deselected="(option) => console.log(option)"
/>
```

::note
Pressing **Backspace** to remove the last tag in multi-select updates `v-model` but does **not** emit `optionDeselected`.
::

### search

Emitted when the search value changes in searchable mode.

| | |
|---|---|
| **Payload** | `string` |

```vue
<Select
  v-model="selected"
  :options="options"
  @search="(query) => fetchRemoteOptions(query)"
/>
```

### menuOpened

Emitted when the dropdown menu opens.

```vue
<Select @menu-opened="onMenuOpened" />
```

### menuClosed

Emitted when the dropdown menu closes.

```vue
<Select @menu-closed="onMenuClosed" />
```

## SelectRoot events

Primitive root emits the same lifecycle events with different selection payloads:

| Event | Payload | Notes |
|-------|---------|-------|
| `update:modelValue` | `SelectModelValue<Value>` | Use with `v-model` |
| `optionSelected` | `OptionValue` | Raw value, not source object |
| `optionDeselected` | `OptionValue \| null` | `null` on multi clear-all |
| `search` | `string` | Search text changed |
| `menuOpened` | | Menu opened |
| `menuClosed` | | Menu closed |

```vue
<SelectRoot
  v-model="model"
  :options="options"
  @option-selected="(value) => console.log(value)"
/>
```
