::info
The assembled `Select` exposes `icon`, `trailing-icon`, `tag-remove`, `no-options`, `create-item`, and `clear` slots. Enable the clear button with the `clearable` prop. For broader customization, compose with [primitives](/guide/primitives).
::

Primitives expose slots for flexible rendering. `SelectRoot`, `SelectTrigger`, `SelectPopover`, and `SelectListbox` also accept a `default` slot for structural composition.

## Assembled Select

The assembled `Select` forwards customization slots to the underlying primitives.

**`tag-remove`**: remove button content for each tag in multi-select mode.

**Slot props:** `SelectTagRemoveSlotProps`

```ts
{ label: string; value: string | number }
```

```vue
<Select v-model="selected" multiple :options="options">
  <template #tag-remove="{ label }">
    <UIcon name="i-lucide-x" :aria-label="`Remove ${label}`" />
  </template>
</Select>
```

**`no-options`**: empty state when search or filtering removes every option.

**Slot props:** `SelectSearchValueSlotProps`

```vue
<Select v-model="selected" :options="options">
  <template #no-options="{ searchValue }">
    No results for "{{ searchValue }}"
  </template>
</Select>
```

**`create-item`**: create action shown when `create-item` is enabled. Receives the trimmed search query.

**Slot props:** `SelectSearchValueSlotProps`

```vue
<Select v-model="value" create-item :options="items" @create="onCreate">
  <template #create-item="{ searchValue }">
    Add "{{ searchValue }}"
  </template>
</Select>
```

**`clear`**: clear button content. Requires `clearable`.

```vue
<Select v-model="selected" clearable :options="options">
  <template #clear>
    Reset
  </template>
</Select>
```

**`icon`**: leading icon before the placeholder or selected value.

```vue
<Select v-model="selected" :options="options" placeholder="Select a country">
  <template #icon>
    <UIcon name="i-lucide-globe" />
  </template>
</Select>
```

**`trailing-icon`**: trailing dropdown icon. Receives open and loading state.

**Slot props:** `SelectTrailingIconSlotProps`

```ts
{ open: boolean; loading: boolean }
```

```vue
<Select v-model="selected" :options="options">
  <template #trailing-icon="{ open, loading }">
    <span v-if="loading">⏳</span>
    <span v-else-if="open">▲</span>
    <span v-else>▼</span>
  </template>
</Select>
```

## Primitives

### SelectValue

Customize how the selected value is displayed.

**Slot props:** `SelectValueDefaultSlotProps<OptionValue>`

```ts
{
  selectedOptions: readonly {
    id: string
    value: OptionValue
    label: string
    disabled: boolean
  }[]
}
```

```vue
<SelectValue placeholder="Pick one">
  <template #default="{ selectedOptions }">
    <span v-if="selectedOptions[0]">
      ✓ {{ selectedOptions[0].label }}
    </span>
    <span v-else class="placeholder">Pick one</span>
  </template>
</SelectValue>
```

Default behavior: single mode shows the first label or placeholder. Multi mode renders `SelectTag` per selection.

**Named slot `tag-remove`:** remove button content for each tag in multi-select mode. Forwards to `SelectTag`'s `remove` slot.

```vue
<SelectValue placeholder="Pick languages">
  <template #tag-remove="{ label }">
    Remove {{ label }}
  </template>
</SelectValue>
```

### SelectOption

Customize option content in the listbox.

```vue
<SelectOption :value="opt.value" :label="opt.label">
  <strong>{{ opt.label }}</strong>
  <span class="text-muted">{{ opt.description }}</span>
</SelectOption>
```

**Named slot `checkmark`:** icon shown on the right when the option is selected in multi-select mode. Only rendered when `multiple` is enabled and the option is selected.

```vue
<SelectOption :value="opt.value" :label="opt.label">
  <template #checkmark>
    <UIcon name="i-lucide-check" />
  </template>
</SelectOption>
```

### SelectGroup and SelectGroupLabel

::note
You do not need group primitives for a flat list. Render `SelectOption` children directly inside `SelectListbox`.
::

```vue
<SelectGroup>
  <SelectGroupLabel>Frontend</SelectGroupLabel>
  <SelectOption value="vue" label="Vue" />
  <SelectOption value="react" label="React" />
</SelectGroup>
```

Both components use a plain `default` slot. `SelectGroupLabel` must be inside `SelectGroup`.

### SelectSeparator

Insert a non-interactive divider between groups or sections.

```vue
<SelectGroup>
  <SelectGroupLabel>Europe</SelectGroupLabel>
  <SelectOption value="FR" label="France" />
  <SelectOption value="DE" label="Germany" />
  <SelectOption value="IT" label="Italy" />
</SelectGroup>

<SelectSeparator />
```

### SelectTag

Customize multi-value tags. Used by `SelectValue` in multi-select mode.

**Default slot props:** `SelectTagRemoveSlotProps`

```ts
{ label: string; value: string | number }
```

**Named slot `remove`:** content for the remove button (default: × icon). Uses the same `SelectTagRemoveSlotProps` shape.

```vue
<SelectTag :value="opt.value" :label="opt.label">
  <template #default="{ label }">
    <span class="tag-label">{{ label }}</span>
  </template>
  <template #remove>
    ×
  </template>
</SelectTag>
```

The remove button calls `context.deselect(value)` and has `aria-label="Remove"`.

### SelectIcon

Leading icon before the value display. Place inside `SelectTrigger`, before `SelectValue`.

```vue
<SelectTrigger>
  <SelectIcon>
    <UIcon name="i-lucide-globe" />
  </SelectIcon>
  <SelectValue placeholder="Pick one" />
  <SelectTrailingIcon />
</SelectTrigger>
```

Rendered with `data-select-icon` and `aria-hidden="true"`.

### SelectTrailingIcon

Trailing dropdown icon for open and loading state.

**Slot props:** `SelectTrailingIconSlotProps`

```ts
{ open: boolean; loading: boolean }
```

```vue
<SelectTrailingIcon>
  <template #default="{ open, loading }">
    <span v-if="loading">⏳</span>
    <span v-else-if="open">▲</span>
    <span v-else>▼</span>
  </template>
</SelectTrailingIcon>
```

Exposed via `data-open` and `data-loading` attributes for CSS-only styling.

### SelectCreateItem

Customize the create action when `create-item` is enabled on `SelectRoot`.

**Slot props:** `SelectSearchValueSlotProps`

```ts
{ searchValue: string }
```

```vue
<SelectRoot v-model="model" create-item searchable :options="options" @create="onCreate">
  <SelectPopover>
    <SelectListbox>
      <SelectOption v-for="option in options" :key="option.value" v-bind="option" />
      <SelectCreateItem>
        <template #default="{ searchValue }">
          Create "{{ searchValue }}"
        </template>
      </SelectCreateItem>
    </SelectListbox>
  </SelectPopover>
</SelectRoot>
```

Default text: `Create "{searchValue}"`. Renders with `role="option"` when visible.

### SelectNoOptions

Customize the empty state when no options match the search filter.

**Slot props:** `SelectSearchValueSlotProps`

```ts
{ searchValue: string }
```

```vue
<SelectNoOptions>
  <template #default="{ searchValue }">
    No languages found for "{{ searchValue }}".
  </template>
</SelectNoOptions>
```

Default text: `"No options"`. Renders with `role="status"` when `filteredOptions` is empty.

### SelectClear

Customize the clear button content. Place inside `SelectTrigger` and enable with `clearable` on `SelectRoot`.

```vue
<SelectRoot v-model="model" clearable :options="options">
  <SelectTrigger>
    <SelectValue placeholder="Pick one" />
    <SelectClear>Reset</SelectClear>
  </SelectTrigger>
</SelectRoot>
```

Visible only when `clearable` is enabled, a value exists, and the select is not disabled. Has `data-select-clear` and `aria-label="Clear selection"`.

### SelectTrigger, SelectInput, SelectListbox, SelectPopover

These accept a `default` slot for child composition but do not expose scoped slot props.

```vue
<SelectTrigger>
  <SelectIcon>
    <UIcon name="i-lucide-globe" />
  </SelectIcon>
  <SelectValue placeholder="Choose" />
  <SelectTrailingIcon />
</SelectTrigger>
```

## TypeScript

Slot names and scoped prop types are exported from `vue3-select-component` for wrappers and custom compositions.

**Assembled `Select`:**

```ts
import type { AssembledSelectSlots } from "vue3-select-component"
```

**Primitives:**

```ts
import type {
  SelectClearSlots,
  SelectCreateItemSlots,
  SelectDefaultSlots,
  SelectGroupLabelSlots,
  SelectGroupSlots,
  SelectIconSlots,
  SelectNoOptionsSlots,
  SelectOptionSlots,
  SelectSeparatorSlots,
  SelectTagSlots,
  SelectTrailingIconSlots,
  SelectValueSlots,
} from "vue3-select-component"
```

**Reusable slot prop types:**

```ts
import type {
  SelectSearchValueSlotProps,
  SelectTagRemoveSlotProps,
  SelectTrailingIconSlotProps,
  SelectValueDefaultSlotProps,
} from "vue3-select-component"
```

`SelectValueSlots` and `SelectValueDefaultSlotProps` accept an optional `OptionValue` generic. `tag-remove` slot props use `value: string | number` so forwarded slots type-check across generic components.

See [TypeScript](/guide/typescript) for props, emits, and model value types.
